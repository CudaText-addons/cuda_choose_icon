plugin for CudaText.
demo plugin, just to show dialog to choose icon file.
a) shows dialog to choose icon set (subdir in icons/ dir in plugin dir)
b) shows 2nd dialog to choose icon from iconset
chosen icon is printed to the Console panel.
size of listbox items is adapted to chosen iconset (16x, 24x, 32x).
